# ShareTogether

一款與好友出遊時便利的線上分帳 App，並提供記事本、留言、推播通知等便利功能

### Features
* 群組管理
    - 新增群組
    - 新增 / 移除成員
* 消費資訊
    - 瀏覽詳細內容
    - 地圖瀏覽
    - 關鍵字搜尋
* 新增消費
    - 提供消費類型、金額、說明
    - 支援均分、比例、金額等分帳方式
    - 消費地點
* 活動紀錄
    - 群組邀請
    - 新增消費通知
